# LibrarySystem
A Library System Using C# Windows Form
圖書館管理系統

*使用技術與工具*
架構：Windows Forms (.NET Framework)
程式語言：C#
資料庫運用：SQL Server
開發工具：Visual Studio


*功能* 
首頁 / 登入與註冊 / 加入書籍 / 查詢書籍
加入學員 / 查詢學員 / 借書 / 還書 / 查詢書籍

*說明*
1.	圖書館管理是個艱鉅的任務，圖書管理員需要大量的記錄管理，因此簡易操作的電子化管理系統可節省大量的時間。
2.	使用Windows視窗介面，操作非常簡易。
